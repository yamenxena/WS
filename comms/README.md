# Communication Samples — Guide

## Purpose
This directory provides tone and style samples for agent-generated communications. The agent reads these to match Majaz's voice — it does NOT copy client-specific data from here.

## Language Tags
- `[AR]` — Arabic content
- `[EN]` — English content
- `[AR/EN]` — Bilingual

## How to Add Samples
1. Paste real examples from WhatsApp, email, or meeting notes
2. **Remove all client names and sensitive data** (replace with [Client])
3. Tag with language
4. Place in the correct file (whatsapp, email, or notes)

## Files
- `whatsapp_samples.md` — informal, quick-response tone
- `email_samples.md` — formal, proposal/municipality tone
- `notes_templates.md` — structured meeting and field notes

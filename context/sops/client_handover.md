# SOP: Client Handover

## Scope
From construction completion through final handover to client. Includes punch list, documentation, and warranty.

## Process

### Step 1: Pre-Handover Inspection
- Comprehensive walk-through with contractor
- Generate punch list: all defects and incomplete items
- Assign severity and deadline for each item
- Deliverable: `Punch List`

### Step 2: Punch List Resolution
- Contractor addresses all items
- Re-inspect each item upon completion
- Close items one by one
- All Critical and Major items must be closed before handover

### Step 3: Documentation Package
Prepare handover folder containing:
- [ ] As-built drawings (architectural + structural + MEP)
- [ ] Municipality completion certificate
- [ ] MEP test certificates (electrical, plumbing, HVAC)
- [ ] Fire safety certificate
- [ ] Waterproofing warranty
- [ ] Equipment manuals and warranties
- [ ] Spare keys and access codes
- [ ] Maintenance schedule

### Step 4: Handover Meeting
- Walk-through with client
- Explain: systems operation, maintenance needs, warranty terms
- Client signs handover acceptance form
- Deliverable: `Signed Handover Form`

### Step 5: Post-Handover
- 30-day defect liability period
- Respond to client-reported issues within 48 hours
- Final close-out after defect liability period

## Quality Checks
- [ ] All punch list items closed (Critical + Major)
- [ ] Documentation package complete
- [ ] Client sign-off received
- [ ] Warranty terms communicated

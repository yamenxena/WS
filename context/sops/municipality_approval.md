# SOP: Municipality Approval

## Scope
Abu Dhabi Municipality submission through NOC/permit issuance. Does NOT cover construction commencement.

## Required Documents

### Architectural
- [ ] Architectural drawings (plans, elevations, sections, site plan)
- [ ] 3D perspectives
- [ ] Area calculations (built-up, FAR, plot coverage)
- [ ] Setback compliance documentation

### Structural
- [ ] Structural drawings (foundation, columns, beams, slabs)
- [ ] Structural calculations
- [ ] Soil investigation report

### MEP
- [ ] Electrical layout + load calculations
- [ ] Plumbing layout + drainage
- [ ] HVAC layout + cooling load
- [ ] Fire safety layout + compliance

### Administrative
- [ ] Title deed (copy)
- [ ] Owner ID (copy)
- [ ] Consultant registration certificate
- [ ] Application form (Abu Dhabi Municipality portal)
- [ ] Fee payment receipt

## Process

### Step 1: Pre-Submission Check
- Verify all documents are complete against checklist above
- Cross-check against current Abu Dhabi codes (`municipality_codes.md`)
- Flag non-compliance items BEFORE submission
- Deliverable: `Pre-Compliance Checklist`

### Step 2: Online Submission
- Submit via Abu Dhabi Municipality e-services portal
- Upload all documents in required format (PDF, max size per file)
- Record submission reference number
- Expected review period: 2-4 weeks

### Step 3: Review Response
- Municipality issues: Approved / Revision Required / Rejected
- If revision: document required changes, revise, resubmit
- Track revision cycles (target: ≤ 2 cycles)

### Step 4: NOC / Permit Issuance
- Collect NOC/building permit
- Distribute copies to client and contractor
- Update project status

## Common Rejection Reasons
1. Setback violations
2. FAR exceedance
3. Missing fire safety compliance
4. Incomplete structural calculations
5. Incorrect area calculations
